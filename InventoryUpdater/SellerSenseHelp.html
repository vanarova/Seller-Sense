Unmapped codes :

Unmapped codes in inventory page, while importing inventory file:

While, importing inventory file, unmapped Amazon, Flipkart, snapdeal codes are shown, these codes are shown based on rule : If a product is found in incoming inventory file, but it is not found in existing in-house inventory, that product code is flagged and its is assumed that mapping in done for this product.

Unmapped codes in inventory page, while importing orders file:

While, importing orders file, unmapped Amazon, Flipkart, snapdeal codes are shown, these codes are shown based on rule : If a product code is found in incoming order file, but it is not available in existing inventory file, it is flagged as unmapped product. Unmapped products in orders should be addressed on priority, as these products can be mapped to any inhouse products, hence these orders are no loaded on inventory page.
